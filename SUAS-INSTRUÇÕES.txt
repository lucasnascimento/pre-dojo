Instruções:

Para montar o projeto no Eclipse será necessário instalar o Lambok
	Instruções em: http://projectlombok.org/download.html
	
Para compilar os fontes baixados via Maven execute no shell: 
	mvn clean package 
	
Para executar o Main execute no shell (usar como argumento seu arquivos de log):
	java -jar target/match-1.0.jar src/test/resources/game2.log
	
Obrigado.
